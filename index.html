경기도교육청 하이러닝 - 교사용 가이드

서비스 소개

교수자와 학습자 간의 맞춤형 교육 플랫폼을 학교 현장의 다양한 요구사항과 필요에 맞춰 교육용 AI 및 학습 관리 시스템 기반의 온라인 교수·학습 플랫폼을 제공합니다. 유연하고 안전한 교육 플랫폼을 제공하여 학교 현장의 편의를 돕겠습니다.

주요 기능

편리한 학급 운영

학급 관리, 출석 관리, 우리반 게시판 등 다양한 학급 운영 기능을 제공합니다.

다양한 수업지원 도구

영상 수업, 화상 수업, 등교 수업 등 다양한 방식의 수업을 지원합니다.

자기주도 학습

학습 콘텐츠, 문제집, 교재 등을 활용하여 학생들의 자기주도 학습을 지원합니다.

AI 맞춤형 진단

AI를 활용한 학습 진단과 리포트로 맞춤형 학습을 지원합니다.

이용 안내

서비스 이용 유의 사항

  • 네트워크 환경을 확인해주세요. 네트워크를 많이 사용하는 프로그램은 종료하시고, 무선 공유기와 가까운 위치에서 사용하시기 바랍니다.
  • 크롬 브라우저 사용을 권장합니다.
  • 화상 수업 시 카메라를 이용하는 다른 프로그램을 모두 종료해주세요.
  • 이어폰이나 헤드셋 사용을 권장합니다.
최초 접속 시 이용 순서
  1. 교사 계정 선택
  2. 회원가입
  3. 로그인
  4. 최초 로그인 시 비밀번호 변경
  5. 내 정보 및 학급 정보 확인
  6. (담임) 우리반 현황 확인
  7. 과목 개설
  8. 수업 진행

회원가입

로그인 > 교사 선택

로그인 영역에서 교사를 선택합니다.

회원가입 선택

'회원가입 하기' 버튼을 선택합니다.

개인정보 이용 동의

이용 약관 및 개인정보 수집 이용에 대한 안내를 확인하고 동의함에 체크합니다.

실명 인증 및 회원정보 입력

'휴대폰 인증' 버튼 선택 시 본인 확인을 위한 인증을 진행합니다.

이름 / 아이디/ 비밀번호 / 이메일 등을 입력합니다.

회원 정보 확인 및 학교 선택

소속된 학교를 검색하고 선택합니다.

회원가입 완료

회원가입을 완료하신 후, '메인 바로가기' 버튼 선택 시 메인 포털로 이동합니다.

주의사항: 회원가입 후 학교 관리자의 승인 후에 로그인 가능하며 담임교사의 경우 관리자페이지에서 담임 배정이 필요합니다.

내 정보 및 학급 정보 확인

본인이 소속된 학교 / 담당 학년, 반 정보를 확인할 수 있습니다.

과목 개설

과목 개설 시 과목과 대상 학급을 지정해야 수업을 원활히 진행할 수 있습니다.

과목 목록 개설 전

'개설하기' 버튼 선택 시 과목을 개설할 수 있습니다.

내가 개설한 과목 / 내가 신청한 과목 탭 메뉴 별로 과목 목록을 조회할 수 있습니다.

과목이 개설된 이후 AI 학습 진단과 리포트 메뉴를 원활히 이용할 수 있습니다.

과목 개설 > 간편 설정

교육과정 / 학년 / 수업 기간 / 과목 / 상세 과목 등을 선택합니다.

위 항목들의 선택 완료 시 "다음으로' 버튼이 활성화 되며, 선택 시 다음 단계를 진행할 수 있습니다.

메인 화면

메인 화면 구성

  • 상단 메뉴 영역 - 나의 교실 / 학습 집단 / 학습 콘텐츠 / AI 리포트 등 다른 메뉴로 이동
  • 알림/할 일 안내 영역 - 최근 알림 및 할 일 목록 제공
  • 사용자 프로필 영역 - 프로필 사진과 정보 확인
  • 오늘의 수업 영역 - 오늘 진행 예정인 수업 정보
  • 담당 과목 학습 현황 영역 - 담당 과목 학습 현황 제공
  • 게시판 영역 - 우리반 게시판과 수업 게시판 내 게시글

나의 할 일 & 알림

나의 할 일

나의 할 일 더보기 버튼 클릭 시 '나의 할 일' 화면으로 이동합니다.

전체 / 수업(질문/답변) / 과제 / 설문 / 문제집 / 진단 탭을 활용하여 접속일 기준 나의 할 일을 조회할 수 있습니다.

알림

상단 알림 아이콘 클릭 > 더보기 클릭 시 '알림' 화면으로 이동합니다.

알림의 종류에 따른 탭 별로 알림을 조회할 수 있습니다.

  • 시스템 관리자가 서비스 관련된 안내 사항을 전달하는 알림
  • 학교 관리자가 학교 전체에 안내하는 알림 사항
  • 고객센터의 1:1 문의 게시판에 게시물을 등록한 후 관리자가 답변을 등록했을 경우 안내되는 알림
  • 비밀번호를 변경할 경우 안내하는 알림
  • 과목이 삭제될 경우 안내하는 알림
  • 학부모가 계정 승인을 요청했을 경우 안내하는 알림

나의 학교

나의 교실

나의 교실 메뉴는 담임 교사 전용 메뉴로 과목 교사에게는 제공되지 않습니다.

나의 교실 기능

  • 담당 학급과 재적 인원을 확인할 수 있습니다.
  • 담당 학급의 보상 레벨 상위 5명인 학생들을 나타냅니다. 클릭 시 우리반 포인트와 랭킹 화면으로 이동합니다.
  • 담당 학급 학생들의 이번주 감정 상태와 출석 현황을 나타냅니다.
  • 학생들의 감정 상태 또는 출석 현황으로 전환하는 버튼을 제공합니다.

학생 이수현황

담당 학급에 소속된 학생과 학생별 이수현황(출석율 / 과제 제출율 / 이수율), 학습 수준을 확인할 수 있습니다.

전학생의 이수현황은 전학 온 당일 기준으로 계산됩니다.

이번주 수업

  • 현재 담당하고 있는 과목 정보를 나타냅니다.
  • 이번주 수업을 확인할 수 있습니다.
  • 수업 설계 전일 경우 수업 설계하기 버튼을 통해 수업 설계화면으로 이동할 수 있습니다.
  • '수업 입장' 버튼 클릭 시 해당 수업 강의실로 입장할 수 있습니다.
  • 이번 주 전체 수업 활동 목록을 통해 제출 현황을 편리하게 확인할 수 있습니다.
