00. 이용 준비

0.1 서비스 소개

경기도교육청 하이러닝 주요 기능

편리한 학급 운영

학급 관리, 출석 관리, 우리반 게시판 등 다양한 학급 운영 기능을 제공합니다.

다양한 수업지원 도구

영상 수업, 화상 수업, 등교 수업 등 다양한 방식의 수업을 지원합니다.

자기주도 학습

학습 콘텐츠, 문제집, 교재 등을 활용하여 학생들의 자기주도 학습을 지원합니다.

AI 맞춤형 진단

AI를 활용한 학습 진단과 리포트로 맞춤형 학습을 지원합니다.

교수자와 학습자 간의 맞춤형 교육 플랫폼을 학교 현장의 다양한 요구사항과 필요에 맞춰 교육용 AI 및 학습 관리 시스템 기반의 온라인 교수·학습 플랫폼을 제공합니다. 유연하고 안전한 교육 플랫폼을 제공하여 학교 현장의 편의를 돕겠습니다.

0.2 이용안내

지원 환경

  • 네트워크 환경 확인 (무선 공유기와 가까운 위치)
  • 크롬 브라우저 사용 권장
  • 쉬운 비밀번호로 설정하지 말기
  • 화상수업 시 카메라를 이용하는 다른 프로그램 종료, 이어폰·헤드셋 사용

최초 접속 시 이용 순서

  1. 교사 계정 선택
  2. 회원가입
  3. 로그인
  4. 최초 로그인 시 비밀번호 변경
  5. 내 정보 및 학급 정보 확인
  6. (담임) 우리반 현황 확인
  7. 과목 개설
  8. 수업 진행

0.3 캐시 삭제

하이러닝 업데이트 이후 정상적으로 진입이 되지 않을 경우 캐시 삭제가 필요합니다.

  1. 크롬 실행 후 더보기 클릭 > 설정 클릭
  2. 개인 정보 보호 및 보안 > 인터넷 사용 기록 삭제
  3. 기간은 지난 4주 이상으로 설정하고, 체크박스는 모두 선택
  4. 데이터 삭제 클릭

캐시를 삭제하면 불필요한 데이터로 인한 서비스 비정상 접근을 해소하고, 저장 공간 부족과 데이터 속도 저하를 예방할 수 있습니다.

01. 시작하기

1.1 회원가입

회원 가입 순서

  1. 로그인 > 교사 선택
  2. 회원가입 선택
  3. 개인정보 이용 동의
  4. 실명 인증 및 회원정보 입력
  5. 회원 정보 확인 및 학교 선택
  6. 회원가입 완료

주의사항: 학교 관리자의 승인 후 로그인 가능. 담임교사의 경우 관리자페이지에서 담임 배정 필요

1.2 로그인

  1. 로그인 > 교사 선택
  2. 아이디/비밀번호 입력
  3. 최초 로그인 시 비밀번호 변경

안전하고 원활한 서비스 이용을 위해 최초 로그인 시 비밀번호를 변경해야 합니다.

1.3 내 정보 및 학급 확인

  1. 프로필 선택 (우측 상단 프로필 영역을 선택)
  2. 편집 아이콘 선택
  3. 내 정보 및 학급 정보 확인 (개인 정보, 담당 학년/반)

1.4 과목 개설

과목 개설 시 과목과 대상 학급을 지정해야 수업을 원활히 진행할 수 있습니다.

  • 과목 목록에서 '개설하기' 버튼 선택
  • 교육과정, 학년, 수업 기간, 과목, 상세 과목 등 선택 후 다음 단계 진행
  • 과목이 개설된 이후 AI 학습 진단과 리포트 메뉴를 원활히 이용할 수 있음

02. 메인 화면

2.1 메인 화면 구성

  1. 상단 메뉴 영역
  2. 알림/할 일 안내 영역
  3. 사용자 프로필 영역
  4. 오늘의 수업 영역
  5. 담당 과목 학습 현황 영역
  6. 게시판 영역

2.2 나의 할 일 & 알림

나의 할 일: 나의 할 일 더보기 버튼 클릭 시 '나의 할 일' 화면으로 이동. 탭을 활용해 조회

알림: 상단 알림 아이콘 클릭 후 더보기 선택 시 알림 화면 이동. 알림 종류에 따른 탭 별로 조회 가능

03. 나의 학교

3.1 나의 교실

담임 교사 전용 메뉴

  • 담당 학급과 재적 인원을 확인
  • 담당 학급의 보상 레벨 상위 5명 확인 (클릭 시 포인트 랭킹 화면 이동)
  • 학생 감정 상태, 출석 현황
  • 이번주 수업 정보, 수업 설계하기, 수업 입장

학생별 이수현황(출석율, 과제제출율, 이수율)과 학습 수준 확인 가능

3.2 나의 수업

  • 개설 과목 - '개설하기' 버튼 클릭 시 과목 개설 가능
  • 내가 개설한 과목 / 참여 수업 목록

3.3 과목 개설 및 관리

과목 상세

  • 과목 수정 아이콘 클릭으로 과목 개설 화면 이동
  • 수업 목록 / 수업 활동 / 게시판 / 수업 질문 / 출결 관리 / 학생 관리 등 조회
  • 수업 설계 버튼으로 수업 설계 화면 이동

수업 활동

과제 / 퀴즈 / 설문 탭 별로 수업 활동 조회

게시판

게시글 목록. 게시글 클릭 시 상세 이동

수업 질문

질문 목록. 클릭 시 질문 상세화면 이동

출결 관리

  • 차시 수업별 출결 상태 상세 조회/변경 가능
  • 등교 수업: 일괄 출석
  • 영상 수업: 시청%에 따라 출석 처리
  • 화상 수업: 입장 시 출석 처리

학생 관리

  • 학생 목록 조회
  • 이수 상태 변경
  • 전학생 자동 등록 처리

3.4 수업 시간표

  • 하루종일 진행 수업은 all-day 영역에 표기
  • 요일별 시간표 확인
  • 수업 설계하기 아이콘 클릭 시 설계 화면 이동

3.5 (담임) 우리반 게시판

  • 검색어와 카테고리를 이용해 게시판 검색
  • 게시글 목록 클릭 시 상세 이동
  • 체크박스 선택 후 삭제 가능
  • 첨부파일 아이콘 클릭 시 파일 다운로드
  • 등록 버튼으로 게시판 등록

전체/학생 그룹별 수신 대상 지정 가능

04. 수업 진행

4.1 등교 수업

수업 목록 – 설계 전

'설계하기' 버튼 클릭 시 수업 설계 화면 이동

수업 설계 – 등교수업

  • '수업 설계안 가져오기' 버튼 선택 시 팝업 제공
  • 등교/화상/영상 중 등교 수업 선택
  • 단원 선택 시 추천 수업 자료 활성화
  • 수업 자료 선택 후 업로드 유형(직접 업로드, 라이브러리 조회 등) 결정
  • '저장' 버튼으로 설계 완료

수업 목록 – 설계 후

수업 시작 20분 전부터 '수업 입장 (N분전)' 버튼 활성화

통합학습창

  1. 수업 준비하기: 교재 미리보기, '수업시작' 버튼
  2. 수업 진행하기: 메인 화면, 필기도구 공유
  3. 발표시키기: 특정 학생 발표자 지정
  4. 질문하고 답하기: 질문한 학생 목록 확인
  5. 교재 등록하기 (파일, 문제풀이, 유튜브, 웹URL, 화이트보드 등)
  6. 함께보기 중 교사 화면과 학생 화면 동기화

4.2 화상 수업

  • '화상 수업' 버튼 클릭 시 화상 수업 방으로 이동
  • 수업 입장 후 화상 수업 진행

4.3 영상 수업

  • '영상 수업' 버튼 클릭 시 동영상 뷰어 팝업
  • 등록된 VOD 시청

05. AI 학습진단

5.1 AI 학습진단

우리반(담임) 또는 과목별 학습 진단 현황 확인

  • 진행중인 학습 진단: 클릭 시 학습 진단 페이지 이동
  • 진단 결과 영역: 과목 별 진단 결과 그래프
  • 학습 진단 목록: 재진단, 결과보기, 현황보기 버튼

5.2 진단하기

  1. '진단하기' 버튼 클릭
  2. AI 학습 진단 실시하기: 응시 시간 시작일/마감일 설정
  3. AI 학습 진단 선택: 유형 4가지 중 랜덤 제공, '학습 진단 생성하기' 버튼
  4. 진단 현황 확인: 재진단 버튼으로 다음 회차 출제

5.3 진단결과

  • 진단 목록: 현황보기/결과보기로 이동
  • 종합 결과/학생별 결과 탭
  • 원 그래프, 막대 그래프, 문제 및 해설보기
  • 학생별 평균 및 과목 평균 비교

5.4 AI 논술진단

  • 국어, 사회, 과학 과목만 제공
  • 논술 문항 출제하기: 학교급, 학년, 과목, 단원 세부 설정
  • AI 자동 채점 결과를 교사가 확인 후 스크롤 조절, 점수 재평가 가능

06. 학습 콘텐츠

6.1 AI 추천 콘텐츠

학생들의 학습 이력을 분석하여 AI가 학습 성취 수준별 콘텐츠를 추천합니다.

  • 추천 콘텐츠
  • 지난 주 가장 많은 관심을 받은 자료
  • 매일 00시 최근 설계 기준으로 수업 설계안 제공
  • 교내 자료 최대 5개 추천

6.2 영상

  • 영상 목록 정렬 기준 (최신순, 인기순)
  • 영상 출처(전체, EBS, e학습터, etc.)
  • 등록하기 버튼으로 영상 만들기 팝업
  • PC 업로드 or URL 등록 (유튜브) or 라이브러리 조회

6.3 교재

  • 등록하기 버튼 클릭 시 교재 등록 팝업
  • 교재 출처 (전체 학교, 우리 학교), 공유 범위 지정
  • 1GB 이하 파일 업로드

6.4 문항

  • 새 문제집 등록하기: 직접 입력, 문항 선택, 랜덤 생성, AI출제, 서술형 문항 등
  • 문제 풀이 결과 확인 및 AI 채점 결과 수정 가능

6.5 내 콘텐츠

제작 / 공유 / 배포 / 저장 탭 메뉴

  • 영상, 교재, 문제집 유형별 목록 조회
  • 편집모드에서 여러 개 선택 후 삭제 가능
  • 배포하기: 과목 선택 후 학생들에게 배포

07. AI 리포트

7.1 과목 리포트

우리반(초/중/고 공통)

  • 이번 학년도 과목별 학습 현황
  • 주요 과목의 종합 진단 결과 (방사형 그래프)

과목별 리포트 영역 – 우리반(중/고)

  • 과목별 학습 수준을 메달 형태로 표시
  • 문제 풀이 수 / 정답률
  • 단원 별 막대 그래프

누적 학습 현황 영역 – 우리반(중/고)

  • 단원별 문제 풀이 수, 정답률
  • 동영상 시청 시간, 난이도별 풀이시간 분석

문제유형 별 정답률 – 우리반(중/고)

  • 개념/응용/심화 문제 별 정답률

과목별 요약 영역

  • 학습 수준/학습량 분석 코멘트
  • 문제 풀이 수, 문제풀이 정답률 수치
  • 최근 진단 단원명과 수준별 메시지, 단원별 학습 수준 그래프

누적 진단 결과 - 과목별

  • 단원별 문제 풀이 수, 정답률, 동영상 시청 시간
  • 개념/응용/심화 난이도별 풀이시간, 정답률

7.2 학생 리포트

학생 분포도 - 우리반

  • 문제풀이 학습 현황 분포도로 확인
  • 학생 그룹별 피드백 보내기

전체 학생 목록 - 우리반

  • 문제풀이 수, 정답률, 월별 문제풀이 학습 변화

학생 분포도 - 과목별

  • 학습 현황 분포도, 그룹별 피드백

전체 학생 목록 - 과목별

  • 문제풀이 수, 정답률, 월별 문제풀이 변화

7.3 종합 리포트

학기 종합 영역 - 우리반

  • 월별 누적 학습 수준 꺾은선 그래프
  • 우리반 학습 수준 진단 결과 (방사형 그래프)
  • 칭찬이 필요한 학생 학습결과

누적 학습 현황 영역 - 우리반

  • 주차별 문제 풀이 수, 정답률
  • 동영상 시청 시간 막대 그래프
  • 개념/응용/심화 난이도별 정답률

학기 종합 영역 - 과목별

  • 월별 누적 학습 수준 꺾은선 그래프와 메시지
  • 칭찬이 필요한 학생 3명 학습 결과

누적 학습 현황 영역 - 과목별

  • 과목 누적 학습 현황

09. 에듀테크 툴

9.1 학급 경영 도구

학급경영도구 진입

우측 하단 플로팅 버튼 > 학급경영도구 진입

우리반 포인트와 랭킹

  • 담당 학급 학생들의 감정/레벨/랭킹 정보
  • 교내/전체 단위로 랭킹 확인 (전일 자정 기준 업데이트)

칭찬포인트

  • 학급/수업 선택 후 학생 선택
  • 스티커 부여, 포인트 추가/삭제 가능

타이머 / 알람 / 초시계

  • 타이머 시간 설정, 일시정지, 중단
  • 알람 시간 설정 후 알람 울리기
  • 초시계 구간 기록

집중벨

벨소리 재생 기능

학생 뽑기 (개별/그룹)

참여 대상 설정, 자리배치 불러오기, 화면공유 기능

학생 섞기

카드 형식으로 학생들을 무작위 배치

룰렛

  • 'START' 버튼 클릭 시 룰렛

자리배치

  • 행/열 개수 설정, 학생 드래그 앤 드롭
  • 이력 저장/불러오기

사다리타기

  • 참여 대상 설정
  • 당첨 항목 1개 이상 필수
  • 결과 전체보기

9.2 클래스보드

클래스보드 진입

우측 하단 플로팅 내 클래스보드 클릭

클래스보드 메인

  • 설계된 수업의 과목 리스트 노출
  • 검색 기능, 최신 차시 정보

과목 폴더 안에 차시 리스트

최신 차시가 가장 앞에 노출. 선택 시 상세로 이동

과목 폴더 안에 차시 상세 정보

  • 공지, 학습주제, 컨텐츠 목록
  • 보드 화면 설정, 수업 결과 보기 가능

수업 결과 보기

  • 문서 교재: 수업 중 실시간 필기, 질문확인
  • 문제풀이 결과(학생별/문항별), 정/오답률
  • 디지털 칠판(화이트보드) 내용
  • YouTube, 웹 URL, Q&A 리스트 확인
